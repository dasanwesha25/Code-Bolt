#🚀 Code-Bolt

Code-Bolt is a modern online code editor built for developers, students, and coding enthusiasts who want to write, run, save, and share code effortlessly.

The idea behind this project was to create a platform where users can experiment with different programming languages, store useful code snippets, and explore solutions shared by the community—all within a clean and intuitive interface.

Whether you're practicing coding problems, testing ideas, or building projects, Code-Bolt provides a smooth and interactive coding experience.

---

## ✨ What You Can Do

- Write code in multiple programming languages
- Run code instantly and view the output in real time
- Save your favorite snippets for future reference
- Share snippets with the community
- Explore and discover code shared by other developers
- Customize the editor theme and font settings
- Create an account and keep your work synced
- Enjoy a responsive and user-friendly interface

## 🎯 Why I Built This

As developers, we often switch between code editors, online compilers, and note-taking apps just to test and save small pieces of code.

Code-Bolt brings these functionalities together in a single platform. The goal was to create a space where coding, learning, and sharing knowledge become simple and accessible.

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/dasanwesha25/Code-Bolt.git
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Open `http://localhost:3000` in your browser to see the application.
